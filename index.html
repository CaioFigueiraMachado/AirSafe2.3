Como o AirShield funciona?

Tecnologia simples e eficiente para proteger o que mais importa

1

Detecção

Sensores avançados monitoram constantemente o ambiente para detectar vazamentos de gás ou fumaça.

2

Alerta

Em caso de detecção, o sistema envia imediatamente um alerta via SMS para seu celular.

3

Ação

O sistema abre automaticamente as janelas para ventilação em caso de vazamento de gás.

Sobre o AirShield

Transformando segurança em tranquilidade!

Somos alunos do 2º ano do ensino médio fazendo o curso do SENAI de Desenvolvimento de Sistemas. O AirShield nasceu como um projeto escolar que se transformou em uma solução real para um problema que afeta muitas famílias.

Nossa missão é criar tecnologias acessíveis que possam salvar vidas e proporcionar tranquilidade para as famílias brasileiras.

Nossos diferenciais:

  • Tecnologia de ponta para proteger sua família
  • Alertas instantâneos e automação inteligente
  • Instalação fácil e suporte dedicado
  • Segurança acessível para todos
